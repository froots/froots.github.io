Automate!

With some upfront time investment, automating your design testing can pay off handsomely over the course of a project.

There are now tools that can test your designs in a wide range of browsers and devices for you, to the specifications that you set.

Automated testing tools can:

  • be set up to run on demand, or automatically when code changes
  • test responsive designs, so that you can set different expectations at different screen dimensions
  • run all these tests in the cloud so you don't need to provide infrastructure
